Phone Number Formatting for India

Your data has numbers like 090586 72704 or 0562 404 0338. The API needs 9190586727040 format. The expression in the body above handles this: removes spaces, removes leading 0, and prepends 91. Always test with your own number first.

Part 2 ‚Äî Step 8
Safe Sending Rules ‚Äî Don't Get Banned
WhatsApp aggressively bans numbers that look like spam. Follow these rules and you'll never have a problem.
  • Max 80 messages per day ‚Äî the workflow stops after 80 regardless of how many leads you have
  • 30 second gap between batches of 5 ‚Äî the Wait node handles this
  • Only message numbers from your scraped data ‚Äî no random numbers
  • Never send the same template twice ‚Äî the WA_Sent column prevents this
  • Use a dedicated number for API ‚Äî not your personal WhatsApp number
  • If someone replies STOP or NO ‚Äî update their Status to "Opted Out" in Sheets and never message again
  • Gradually scale ‚Äî start with 20/day for the first week, then 40, then 80. This builds sender reputation.
  • Personalize every message ‚Äî the {{1}} and {{2}} variables in the template do this automatically
What Gets You Banned

Sending 200+ messages/day from a new number. Sending identical messages to many numbers at once. Getting reported by recipients. Using a personal WhatsApp number (not the Cloud API number). Always use the API with a proper business number ‚Äî not your SIM card WhatsApp.
