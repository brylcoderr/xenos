PHASE 01 Payment & Agreement 0/4
✓
50% deposit invoice sent and paid
Send invoice via Wave. Work does NOT begin until deposit clears. Confirm payment received via bank/UPI/Razorpay notification.
CRITICAL — NO EXCEPTIONS
✓
Client contract signed
Send the 1-page Xenotrix Service Agreement. Get a WhatsApp "I agree" or email confirmation as minimum. PDF signature preferred.
COLLECT
✓
Project scope confirmed in writing
Send the accepted proposal back to client. Ask them to reply "Confirmed" on WhatsApp. This becomes your scope protection. Screenshot it.
ACTION
✓
Project added to Notion tracker
Create new project page in Notion: Client name, start date, deadline, deal value, GitHub repo link, staging URL placeholder.
ACTION
PHASE 02 Content Collection 0/6
✓
Business logo received (PNG + SVG if possible)
Ask for logo on white background AND transparent background. If they only have JPG, use remove.bg (free) to extract it.
COLLECT
✓
Brand colors and fonts confirmed
Ask: "What are your brand colors? Do you have a hex code? What font do you use?" If they don't know — look at their current site or existing materials and match it.
COLLECT
✓
All page text / copy received
Send the content brief: "Please send text for Home, About, Services, and Contact pages." Set a 48-hour deadline. If they delay, project timeline shifts — document this.
COLLECT
✓
All images / photos received
Business photos, product images, team photos. If they have none, use Unsplash or Pexels (free). Tell them in advance so there are no surprises.
COLLECT
✓
Domain name and hosting access confirmed
Ask: "Do you own a domain? If yes, what is it and can you add me as a DNS manager?" If no domain — register one via Cloudflare Registrar (cheapest, ~$9/year).
ACTION
✓
WhatsApp number and contact details for the website confirmed
Phone number for click-to-call, WhatsApp number, email for contact form, physical address (if needed), Google Maps embed link, social media URLs.
COLLECT
PHASE 03 Technical Setup 0/6
✓
GitHub repo created for this project
Create private repo: xenotrix/[client-name]-website. Add your partner dev as collaborator. Commit initial project structure Day 1.
ACTION
✓
Vercel project created and connected to GitHub
Deploy to Vercel immediately. Share staging URL with client on Day 2. This shows progress and builds trust early.
ACTION
✓
Google Analytics 4 account created for client
Create GA4 property. Add the tracking code to the site. Share access with client's Gmail. This is a standard deliverable they never think to ask for but always appreciate.
ACTION
✓
Google Search Console set up
Add property for their domain. Verify via DNS. Submit sitemap after launch. This is free SEO value you add at zero cost.
VERIFY
✓
Contact form tested and working
Submit a test form. Check the submission arrives on the client's email. Check it works on mobile. This is the #1 thing broken on most sites — fix it first.
VERIFY
✓
SSL certificate active on domain
Vercel provides free SSL automatically. Confirm https:// works and there is no browser warning. Never launch a site without SSL — Google penalizes it.
VERIFY
PHASE 04 Handoff & Retainer Pitch 0/6
✓
Final balance invoice (50%) sent and paid
Send remaining 50% invoice on launch day. Site goes live AFTER payment clears OR both parties agree on DNS transfer timing. Never skip this.
CRITICAL
✓
All login credentials handed to client securely
Create a Google Doc or Notion page with all passwords, login URLs, and access details. Share securely. Never send passwords over WhatsApp as plain text.
SEND
✓
2-minute walkthrough video recorded and sent (Loom)
Record a Loom video walking through the finished site. Show them every feature. This reduces support questions by 80% and makes clients feel taken care of.
SEND
✓
Testimonial / review requested
On the launch call say: "We are a growing agency and reviews mean everything to us. Would you be happy to send a quick 2-sentence WhatsApp message about your experience? We will quote it on our website."
ACTION
✓
Maintenance retainer pitched on launch call
"We can keep your site fast, secure, and updated for $200/month — we monitor uptime, do monthly updates, and you get priority support. Want us to just handle it?" Say this on every launch. 40% will say yes.
ACTION
✓
Referral ask made
"If you know any other business owners who need a website or need to improve theirs — we give you 10% of whatever they spend as a thank-you. No pressure, just wanted you to know."
ACTION
